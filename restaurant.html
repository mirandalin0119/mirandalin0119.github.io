Restaurant Finder

Introduction

In this assigment done for my Web Development Class, my group created a website for users to find restaurants according to their preferences, or according to their location. We took a mobile-first approach to make the website responsive for different screen sizes. Our website was designed and implemented froms scratch using HTML, CSS and JavaScript. We also used Bootstrap components, Mapbox, as well as Zomato's API for this project.
